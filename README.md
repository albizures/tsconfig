# Typescript config

> Shared Typescript config for all my projects

## Installation

```bash
  yarn add @albizures/tsconfig
```

then in `tsconfig.json`

```json
{
	"extends": "@albizures/tsconfig/tsconfig.json",
	"compilerOptions": {
		"outDir": "dist"
	}
}
```
